# 📁 Carpeta de Alumnos

Cada alumno debe crear su propia subcarpeta aquí con el formato `nombre-apellido`.

## Ejemplo de estructura

```
/alumnos
    ├── ana-garcia/
    │   ├── leccion1/
    │   ├── leccion2/
    │   ├── leccion3/
    │   ├── leccion4/
    │   ├── leccion5/
    │   ├── ABP_7/
    │   └── README.md
    └── juan-perez/
        └── ...
```

## Instrucciones

1. Crea tu carpeta: `alumnos/tu-nombre-apellido/`
2. Sube tus entregas en subcarpetas por lección
3. Incluye un `README.md` con descripción de tu proyecto
4. Consulta [GIT_WORKFLOW.md](../GIT_WORKFLOW.md) para el flujo correcto

> ⚠️ Solo modifica **tu propia carpeta**.
