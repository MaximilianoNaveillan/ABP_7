## 🎯 Proyecto Integrador: Vue Product Showcase

Una **SPA (Single Page Application)** que funciona como catálogo interactivo de productos para un departamento de e-commerce.

### ¿Qué construiremos?

| Lección | Objetivo |
|---------|----------|
| L1 | Estructura base con componentes Vue y ciclo de vida |
| L2 | Consumo de API REST con Axios |
| L3 | Gestión de estado global con Vuex |
| L4 | Pruebas unitarias (Jest/Vitest) y E2E (Cypress) |
| L5 | Librería UI (Vuetify/Element Plus) + diseño responsive |
